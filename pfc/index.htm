Introduction

    This program is made to control the pressure of a system. It can also be used in control temperature, water, or any other thing similar.

    Using Fuzzy Logic technology makes program runs better, more stable and more powerful. Rules in this program is:

  • If(muc is dung_muc)then(valve is giu_nguyen)
  • If(muc is thap_hon)then(valve is bom_nhanh)
  • If(muc is cao_hon)then(valve is xa_nhanh)
  • If(muc is dung_muc)and(tocdo is duong)then(valve is xa_cham)
  • If(muc is dung_muc)and(tocdo is cham)then(valve is bom_cham)

    Using Simulink Toolbox makes program clearer and easier to understand, to modify or so on

    Graphic User Interface makes Users fell more please and comfortable

System required

  • Matlab 6.0 or higher installed
  • Simulink and Fuzzy Toolbox required

How to use

    First
  • Open your version of Matlab
  • Change your current directory into the directory contains the program
  • Type apsuat
    Or
  • Simply double-click the file apsuat.mdl 
    Then
  • Specify the data for the system
  • Press Start
